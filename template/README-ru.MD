# Задача шаблон

У Семёна есть две сумки. В первой сумке A яблок. Во второй B груш. A,B(0<=A,B<=2^32).
Сколько у Семёна всего фруктов?

Формат выходных данных

Выведите одно натуральное число — сумму фруктов.

Замечание

Фруктов может быть 0.

Примеры данных

Пример 1
in:
2 2
out:
4
Пример 2
in:
0 0
out:
2
