# Задача Посчитать тарифф

Человек платит деньги A за месяц, где лимит по мегабайтам B, при этом если происходит превышение лимита он должен платить за каждый мегабайт C.
Планирует же загрузить D мегабайт.
Найти сумму.


### Замечание

Пример 1

in:
```
100 10 12 15
```
out:
```
160
```

Пример 2

in:
```
100 10 12 1
```
out:
```
100
```
