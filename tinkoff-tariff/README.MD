# Task Calculate tariff

A person pays money A for a month, where the limit is per megabyte B, and if the limit is exceeded, he must pay for each megabyte C.
Plans to download D megabytes.
Find the amount.


### Notice

Example 1

in:
```
100 10 12 15
```
out:
```
160
```

Example 2

in:
```
100 10 12 1
```
out:
```
100
```
